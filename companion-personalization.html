Конкретные фичи персонализированных советов

4.1 Анализ транзакций и категоризация

Гибридный pipeline: правила → векторный поиск → ML → LLM fallback.

УровеньТехнологияТочностьЛатентность
1. ПравилаRegex/keyword matchingВысокая для известныхНулевая
2. Векторный поискEmbeddings (OpenAI/pgvector)~95%~10ms
3. ML-модельBERT/DeBERTa fine-tuned~98%~50ms
4. LLM fallbackGPT-4o-mini / локальная~99%~500ms
Enrichment. «POS DEBIT SQ *BLUE BOTTLE COF» → «Blue Bottle Coffee, Сан-Франциско, $4.50, Категория: Еда». Moneyhub показывает 98% точность категоризации.

09

Выводы

1

Компаньон отстаёт от лидеров. MBank и О!Bank опережают по глубине цифрового опыта, но у Компаньона прочная база: 1+ млн пользователей, 500+ сервисов.

2

Персонализация = выживание. 74% потребителей ожидают персонализированного опыта. Банки без этого теряют конкурентоспособность.

3

Начать можно просто. Transaction categorization + spending insights на rules + embeddings — 1–2 месяца, 1–2 разработчика. Эффект виден сразу.

4

Регуляторика благоприятна. Цифровой сом, ELQR, открытые API — создают среду для digital-first подхода.

5

Монетизация прозрачна. Внутренняя конверсия + referral fees + embedded marketplace.

6

Moat = данные. Чем раньше начать собирать behavioral data, тем сильнее конкурентное преимущество.
